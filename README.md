# Rede-Neural-para-Classifica-o-Multiclasse-de-Sementes
Este projeto implementa, do zero, uma Rede Neural Artificial simples para resolver um problema de classificação multiclasse utilizando dados reais.

A aplicação foi desenvolvida com o objetivo de aprender padrões a partir de atributos numéricos e classificar diferentes categorias de forma automatizada, sem o uso de bibliotecas prontas de machine learning.

O modelo foi treinado com dados contendo características geométricas, como área, perímetro, comprimento e assimetria, permitindo distinguir entre diferentes classes com base nesses atributos.

Durante o desenvolvimento, foram implementados conceitos fundamentais de aprendizado de máquina, incluindo:

- Normalização de dados (padronização)
- Codificação one-hot para classes
- Forward pass
- Função de ativação sigmoide
- Treinamento com gradiente descendente
- Avaliação com métricas de classificação

## O desempenho do modelo foi analisado utilizando:

- Acurácia
- Precisão (Precision)
- Recall
- F1-Score
- Matriz de confusão

Os resultados demonstram que mesmo uma arquitetura simples é capaz de capturar padrões relevantes em dados bem estruturados, alcançando alto desempenho na tarefa de classificação.

## Tecnologias utilizadas
- Python
- NumPy
- Implementação manual de rede neural
